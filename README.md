# 📈 StockSenseAI

**StockSenseAI** is an AI-powered platform designed to simplify stock market analysis and insights. It provides intelligent tools for stock screening, trend detection, and personalized financial insights.

---

## 🚀 Features

- 🔍 **Stock Screening** – AI-powered stock filtering and recommendations
- 📊 **Real-Time Data** – Live market data and charts
- 🧠 **ML Models** – Forecasting trends and stock movement using ML
- 🎯 **User Portfolio** – Manage watchlists and portfolios
- 🧾 **News & Sentiment Analysis** – Real-time news and sentiment scoring
- 🗣️ **Voice & Chat Interface** – Chatbot/Voice assistant to query stock info

---

## 🛠️ Tech Stack

- **Frontend**: React.js / Next.js
- **Backend**: FastAPI / Python
- **AI/ML**: Mistral AI / OpenAI / Custom ML Models
- **Database**: PostgreSQL / MongoDB
- **Hosting**: Vercel (Frontend), Render/AWS (Backend)
- **CI/CD**: GitHub Actions / GitLab CI
- **Others**: Docker, Redis, WebSockets

---

## 🧪 Running Locally

### 1. Clone the repo

```bash
git clone git@github.com:GaneshajDivekar/StockSenseAI.git
cd StockSenseAI
```

### 2. Start Backend (FastAPI)

```bash
cd backend
uvicorn main:app --reload
```

### 3. Start Frontend (Next.js or React)

```bash
cd frontend
npm install
npm run dev
```
